В мире, где земля проклята, цивилизация выживает либо в движении, либо за непреступными стенами.

---

# Беррондар

_Оплот камня_

![[img/Berrondar.png]]

> _«Пусть они бегут. Горы стояли здесь до нас, и будут стоять после нас. Мы никуда не уйдём»._

|                                 |                                                |
| ------------------------------- | ---------------------------------------------- |
| **Тип**                         | Статичный город-крепость                       |
| **Расположение**                | Горная гряда «Зубы дракона», восток континента |
| **Рекомендуемые Происхождения** | _Великан, Полурослик, Фирболг_                 |
| **Рекомендуемые Сообщества**    | _Горное, Подземное, Сельское_                  |

**Тип:** статичный город-крепость.
**Расположение:** горная гряда «Зубы дракона», восток континента.
**Рекомендуемые Происхождения:** _Великан, Полурослик, Фирболг_
**Рекомендуемые Сообщества:** _Горное, Подземное, Сельское_

### Описание

Беррондар — единственный город, отказавшийся от движения. Он зажат в естественном «кармане» неприступных скал. Единственный открытый проход перекрыт **Великой стеной** — циклопическим сооружением из чёрного гранита и стали, которое служит одновременно и фортификацией, и портовым доком для воздушных судов.

### Экономика: [[Ресурсы#💎 Хрусталь|Хрусталь]]

- **Экспорт:** город сидит на богатейших жилах магического хрусталя.
- **Импорт:** продовольствие и древесина (в горах с этим туго).

### Устройство

- **Стена (порт):** здесь кипит торговля. Чужаков редко пускают дальше Стены.
- **Город:** жилые кварталы разной степени достатка, в центре — высоченная башня, сохранившая историческое название «Императорская».
- **Шахты:** уходят глубоко под землю и в толщу гор. Там тепло, душно и всегда слышен стук кирок.

**Слух:** говорят, шахтеры докопались до чего-то _живого_ в корнях гор. И теперь Беррондар не просто добывает хрусталь, а сдерживает что-то, что хочет выбраться наружу.

# Зефир

_Небесная гавань_

![[img/Zephyr.png]]

> _«Смотри под ноги. Здесь первый шаг — это прогулка, а второй — падение с двух километров»._

|                                 |                                                              |
| ------------------------------- | ------------------------------------------------------------ |
| **Тип**                         | Летающий город, архипелаг платформ                           |
| **Расположение**                | Дрейфует по высотным воздушным потокам, охотится за штормами |
| **Рекомендуемые Происхождения** | _Катари, Симайя, Аэрис_                                      |
| **Рекомендуемые Сообщества**    | _Благородное, Научное, Морское_                              |

### Описание

Сверкающий шпилями город из стекла, латуни и белого камня. Он удерживается в воздухе благодаря колоссальным антигравитационным двигателям и помощи пришвартованных кораблей. Зефир высокомерен: его жители буквально смотрят на остальных свысока.

### Экономика: [[Ресурсы#⚡ Молнии|Молнии]]

- **Экспорт:** Энергия (заряженные аккумуляторы), высотные технологии.
- **Побочный продукт:** грязный Эфир («Грозовой осадок»).
- **Налог на гравитацию:** любое судно при стыковке обязано держать свои антигравы включенными, помогая городу висеть в воздухе.

### Устройство

- **Верхний Шпиль:** жилища аристократии, обсерватории и залы Совета.
- **Пена:** зона названа так из-за скопления шаров с летучим газом, которые держат постройки простых работяг.
- **Грозовые фермы:** нижние уровни, утыканные гарпунами-молниеотводами. Самое опасное место, где работают «ловцы гроз».

**Слух:** ненадёжные источники сообщают, что двигатели Зефира изношены. Город медленно теряет высоту, а Совет скрывает это, требуя от посетителей всё большей мощности при стыковке.

# Фальмут

_Живая Цитадель_

![[img/Falmouth.png]]

> _«Не ломай ветку, на которой стоишь. Она может обидеться и сбросить тебя»._

|                                 |                                                      |
| ------------------------------- | ---------------------------------------------------- |
| **Тип**                         | Шагающий город-растение                              |
| **Расположение**                | Мигрирует по плодородным зонам и руслам высохших рек |
| **Рекомендуемые Происхождения** | _Галапа, Фунгрил, Фейри_                             |
| **Рекомендуемые Сообщества**    | _Кочевое, Лесное, Сельское_                          |

### Описание

Город, расположенный на ветвях и внутри ствола исполинского Древа, которое медленно переставляет свои корни-ноги. Дома здесь не строят, а _выращивают_. Магия природы здесь так сильна, что металл ржавеет быстрее, а раны заживают за часы.

### Экономика: чистый [[Ресурсы#💧 Эфир|Эфир]]

- **Экспорт:** **«слёзы Древа»** (эфирная роса). Самый чистый и дорогой Эфир в мире.
- **Импорт:** редкие минералы, технологии (с неохотой).

### Устройство

- **Крона:** верхушка древа, где ведётся сбор росы и располагаются элитные жилища друидов.
- **Ветви:** многочисленные толстые отводы, на которых держатся дома, общественные места и пристани для кораблей.
- **Корни:** двигательный отсек. Здесь био-инженеры стимулируют нервные узлы Древа, чтобы оно шло в нужном направлении.

**Слух:** поговаривают, что древо умирает или мутирует. Роса стала горчить, а в нижних корнях завелись паразиты, похожие на людей, сросшихся с корой.

# Солетад

_Ледокол_

![[img/Soletad.png]]

> _«Холод сохраняет. Холод закаляет. Слабаки здесь становятся кормом для рыб»._

|                                 |                                                            |
| ------------------------------- | ---------------------------------------------------------- |
| **Тип**                         | Цилиндрический город на шести огромных металлических ногах |
| **Расположение**                | Замерзшие моря и тундра Севера                             |
| **Рекомендуемые Происхождения** | _Дварф, Дракона, Орк_                                      |
| **Рекомендуемые Сообщества**    | _Догматичное, Морское, Морозное_                           |

### Описание

Массивный город, обшитый корабельной бронёй и шкурами чудовищ. Он передвигается на шести гигантских и очень сложных конечностях в поисках рек и водоёмов, на которые садится своим цилиндром, разламывая и растапливая лёд для добычи рыбы. Воздух здесь пахнет ворванью, дымом и солью. Технологии грубые, но безотказные (паровые машины, гидравлика).

### Экономика: провизия и секретный [[Ресурсы#Эфир|Эфир]]

- **Экспорт:** рыба, мясо морских чудовищ, шкуры.
- **Спец-ресурс:** **Ворвань Левиафана** (Эфир из жира китов). Солетад _не продает_ его, используя для обогрева города и питания своих мощных ледокольных машин.

### Устройство

- **Внешний круг:** чем ближе к наружней стороне — тем холоднее.
- **Генератор:** сердце города, где сжигают жир и кости. Самое теплое место, доступ куда нужно заслужить.
- **Низы:** зона добычи, самое опасное место, где жители пытаются поймать рыбу и не дать рыбе поймать себя.

**Слух:** капитаны Солетада знают безопасный путь через ледяные пустоши к «Оазису» — мифическому месту с горячими источниками, где безопасно, тепло и растут экзотические растения. Но они убивают любого, кто пытается следить за их навигаторами.

# Вангелос

_Железный хищник_

![[img/Vangelos.jpg]]

> _«Мир — это ресурс. И мы его добудем»._

|                                 |                                                 |
| ------------------------------- | ----------------------------------------------- |
| **Тип**                         | Гусеничный мегаполис-хищник                     |
| **Расположение**                | Где угодно. Обычно там, где есть кем поживиться |
| **Рекомендуемые Происхождения** | _Гоблин, Кланк, Гном_                           |
| **Рекомендуемые Сообщества**    | _Научное, Криминальное, Военное_                |

### Описание

Вершина инженерной мысли и жестокости. Огромная пирамида из дымящих труб, заводов и жилых блоков, поставленная на колоссальных размеров гусеничные траки.
Вангелос не торгует — он присоединяет (поглощает) мелкие поселения, разбирая их на запчасти и рекрутируя население.

### Экономика: война и производство

- **Потребление:** жадно скупает (или отбирает) Молнии и металл.
- **Экспорт:** оружие, тяжёлая техника, наёмники.

### Устройство

- **Челюсти (фронт):** Гигантские манипуляторы и пилы для разбора препятствий и захвата других городов.
- **Траки (нижний город):** Трущобы, где живут рабочие и пленники, обслуживающие ходовую часть в ядовитых испарениях.
- **Капитанский мостик:** центр управления Технократов.

**Слух:** Вангелос ищет не просто ресурсы. Его курс проложен по следам древних довоенных бункеров. Технократы ищут чертежи оружия «Судного дня».
